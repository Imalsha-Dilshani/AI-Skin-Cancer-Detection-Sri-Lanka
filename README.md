# AI-Skin-Cancer-Detection-Sri-Lanka
The Use of Artificial Intelligence for Early Detection of Skin Cancer in Sri Lanka

## Project Description
This project aims to develop an Artificial Intelligence (AI) model for early detection of skin cancer through dermoscopic images of skin lesions. The study seeks to evaluate the AI models based on datasets related to the Sri Lankan population.

## Objectives
- Detection of skin cancer using AI technique.
- Compare the proposed algorithm with the baseline algorithm.
- Assess the performance of the system.
